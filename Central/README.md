# Empiria Central Framework

Empiria Core provides the following types:

-  Accounts
-  Authorization
-  Calendars
-  Contracts
-  Documents
-  Events
-  Electronic Sign
-  Geography
-  Measurement
-  Money
-  Parties
-  Postings
-  Procedures
-  Products
-  Projects
-  Protocols
-  Relations
-  Roles
